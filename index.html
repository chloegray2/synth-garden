ðŸ§  How It Works

  1. The potentiometer simulates light levels for the digital plant.
  2. The Arduino reads the analog value and sends it over serial.
  3. The Node.js server relays this data to the browser using socket.io.
  4. p5.js visualizes plant growth based on the input.
  5. Tone.js plays audio cues for healthy, dangerous, and win/lose states.
